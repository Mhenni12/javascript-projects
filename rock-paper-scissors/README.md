# Rock Paper Scissors Web App (HTML, CSS & JavaScript)

> **Note:** This project was built on July 16th 2024 as one of my first web apps using JavaScript. It allows users to play Rock Paper Scissors against the computer.

![RPS Demo](./Demo-rock-paper-scissors.png) 

## Project Overview

This project is a **Rock Paper Scissors game** built using **HTML, CSS, and JavaScript**. It includes:

- Interactive **buttons for Rock, Paper, and Scissors**  
- Displays **player choice, computer choice, and the winner**  
- **Reset button** to start a new game  
- **Autoplay button** to let the computer play automatically  
- **Responsive design** suitable for desktop and mobile screens  

This project helped me practice **JavaScript logic, event handling, and DOM manipulation**.

## Features

- Play Rock Paper Scissors against the computer  
- Reset the game anytime using the **Reset button**  
- Enable **Autoplay mode** for automated gameplay  
- Score tracking for both player and computer  
- Responsive design with a clean, simple UI  

## Technologies Used

- **HTML5** – for structuring the webpage  
- **CSS3** – for styling and layout  
- **JavaScript** – for game logic and interactivity  
- **DOM Manipulation** – updating game state and scores dynamically
