# To-Do List Web App (HTML, CSS & JavaScript)

> **Note:** This project was built on July 30th as one of my first JavaScript web apps. It allows users to manage their tasks with a simple and interactive interface.

![ToDo Demo](./Demo-to-do-list.png) 

## Project Overview

This project is a **To-Do List app** built using **HTML, CSS, and JavaScript**. It includes:

- Add, edit, and delete tasks  
- Mark tasks as completed  
- Responsive design for both desktop and mobile screens  

This project helped me practice **DOM manipulation, event handling, and dynamic content updates using JavaScript**.

## Features

- Add tasks to your to-do list  
- Mark tasks as completed and uncompleted  
- Delete individual tasks  
- Responsive layout for all screen sizes  
- Smooth and interactive user interface  

## Technologies Used

- **HTML5** – for page structure  
- **CSS3** – for styling and layout  
- **JavaScript** – for dynamic task management  
- **DOM Manipulation** – handling task creation, completion, and deletion
