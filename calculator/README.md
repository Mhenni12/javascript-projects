# Calculator App (HTML, CSS & JavaScript)

> **Note:** This project was built on July 7th 2024 as my first web project after learning JavaScript. It is a fully functional calculator that performs basic arithmetic operations.

![Calculator Demo](./Demo-calculator.png)

## Project Overview

This project is a **calculator app** built using **HTML, CSS, and JavaScript**. It includes:  

- A **user-friendly interface** with buttons for digits and operations  
- Basic arithmetic operations: addition, subtraction, multiplication, division  
- **Clear (C) functionaly**  
- Responsive design that works on desktop and mobile devices  

This project helped me practice **DOM manipulation, event handling, and dynamic updates with JavaScript**.


## Features

- Basic arithmetic operations (+, -, ×, ÷)  
- Clear button  
- Responsive design  
- Fully interactive with **JavaScript event handling**  

## Technologies Used

HTML5 – for the structure of the page

CSS3 – for styling and layout

JavaScript – for interactivity and calculations
